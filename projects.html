Projects

IBEX Processor — RTL-to-GDSII

Implemented the RISC-V IBEX core through OpenLane to GDSII with zero DRC/LVS and zero timing violations. Reduced die area by 42%, decreased clock skew by 1ns, and achieved ~5% power reduction.

Tools: OpenLane, NGSpice, KLayout

Tuner — Real-time Note Detection

Correlation-based musical note detection in SystemVerilog with ~99% accuracy. Used fixed-point arithmetic for ~40% compute reduction and pipelined architecture for continuous 44.1kHz processing.

Tools: SystemVerilog, Verilator

DinoRun — FPGA-based Chrome Dino Game

Implemented the classic Chrome Dino Run game in Verilog for FPGA. Designed VGA graphics controller for game rendering and integrated real-time keyboard inputs. Achieved smooth gameplay with obstacle spawning, collision detection, and score tracking on hardware.

Tools: Verilog, ModelSim, Vivado, FPGA (Basys3)

StopIt — FPGA Reaction Timer Game

Developed a reaction-time game on FPGA using SystemVerilog. Implemented random delay generator with LFSR, LED shifting, and 7-segment display drivers. Verified functionality through simulation and synthesized on Basys3 board with Yosys and Vivado.

Tools: SystemVerilog, Verilator, Yosys, Vivado, GTKWave
